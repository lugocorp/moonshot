Install MoonShot

How to install

Moonshot is supposed to be simple to install. It has no external dependencies, so you'll only have to run the build script provided with the source code.

Step 1 to installing Moonshot is to download it. You'll want to go to the GitHub page and download the repository. Once you've cloned or downloaded the project, you can run the build script.

The Moonshot project uses a Makefile for the build system. To build the project, simply run make moonshot. If you'd like to install the command-line interface in your $(HOME)/bin folder, run make install.

Once this command has run successfully, you can use the moonshot command from the terminal. If you find you can't use the command, you may need to add $(HOME)/bin to your PATH variable.
